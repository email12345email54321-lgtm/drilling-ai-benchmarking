# Исследование возможностей ИИ-ассистентов в задачах бурения
**Автор:** Лодин К., группа БНГа-о24 (ЮРГПУ (НПИ))

## 1. Цель работы
Провести сравнительный анализ функциональности зарубежного (Gemini) и отечественного (YandexGPT) чат-ботов. Смоделировать сценарии использования для нефтегазового сектора (бурение скважин) и оценить эффективность каждого бота в решении инженерных задач.

## 2. Структура репозитория
* `/cases` — описание выбранных проблемных ситуаций (кейсов).
* `/prompts` — исходные тексты запросов, отправленных нейросетям.
* `/results` — скриншоты ответов обеих моделей.
  * `/foreign_bot` — ответы Gemini (Google).
  * `/domestic_bot` — ответы YandexGPT (Yandex).
* `/analysis` — подробный сравнительный анализ и выводы.

## 3. Описание кейсов
Для проверки способностей ИИ были выбраны две критические области в бурении:
1. **Кейс №1: Оптимизация механической скорости (ROP).** Анализ причин падения скорости проходки и появления осевых вибраций при бурении глин на глубине 2600 м.
2. **Кейс №2: Ликвидация дифференциального прихвата.** Разработка алгоритма освобождения бурильной колонны с учетом правил безопасности РФ.

## 4. Методология
* Сформулированы единые инженерные запросы для обоих сервисов.
* Запросы отправлены в новых чатах для исключения влияния контекста.
* Ответы зафиксированы в виде скриншотов и проанализированы по критериям технической точности и соответствия отраслевым стандартам.

## 5. Основные выводы
* **Gemini (Google):** Показал превосходство в технической логике. Бот детально объяснил физику резонансных колебаний КНБК и предложил верное решение по изменению частоты вращения.
* **YandexGPT (Yandex):** Продемонстрировал лучшее знание российских регламентов. Бот корректно указал на необходимость оформления нарядов-допусков, специфику установки нефтяных ванн и требования к СИЗ согласно правилам безопасности в РФ.
